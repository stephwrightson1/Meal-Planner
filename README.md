# The Recipe Vault

A personal digital cookbook for GitHub Pages.

Includes 17 recipe cards: Breakfast B01-B05, Lunch L01-L03, Dinner D01-D04, Fakeaway F01, Snacks S01-S02, Sweet Treats C01-C02.

Upload all files and the full assets folder to your GitHub repository, then commit changes. GitHub Pages should use branch main and folder /(root).
